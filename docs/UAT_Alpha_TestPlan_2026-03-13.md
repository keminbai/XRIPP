## 0. 当前状态修正（2026-03-13 运行时复核）

基于 2026-03-13 晚间真实运行日志复核，本文件原先的“可以启动第一轮 Alpha UAT”结论已不再成立，原因不是页面细节，而是运行基线尚未收敛：

- `POST /v3/admin/members` 在运行环境返回 `Request method 'POST' is not supported`
  - 说明当前运行中的后端实例没有加载仓库内最新的会员 CRUD 代码
- `GET /v3/admin/contents` 运行时 SQL 报错 `invalid column 'extra_json'`
  - 说明当前运行库缺少 `docs/DDL_Phase18_Contents_ExtraJson.sql`
- `GET /v3/partner/publishes` 运行时 SQL 报错 `invalid object name 'activity_records'`
  - 说明当前运行库缺少 `docs/DDL_Phase17_ActivityRecords.sql`

因此，本文件现阶段只能作为“恢复运行基线后的测试范围清单”，不能再作为“当前环境可直接开测”的依据。

## 8. 缺陷分级建议

### 阻塞级

1. 关键流程无法提交
2. 支付后状态错误
3. 审核后前台不展示
4. 数据未落库或错落库
5. 权限越权

### 高优先级

1. 列表/导出口径不一致
2. 附件无法查看
3. 状态机错误
4. 数据串角色/串 partner

### 中优先级

1. 文案错误
2. 样式错位
3. 空态/提示不准确
